# RCV-Website
Hugo-based website for Radio Communication Volunteers of St. Joseph County, Indiana  

## Website Prerequisites

* [Install Hugo](https://gohugo.io/installation/)

```sh
winget install Hugo.Hugo.Extended
```

* [Install Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
* [Install Go](https://go.dev/doc/install)
* [Install Dart SASS](https://sass-lang.com/install/)

```sh
npm install -g sass
```
  
* [Install Powershell](https://learn.microsoft.com/en-us/powershell/scripting/install/install-powershell-on-windows?view=powershell-7.6)

```sh
winget install Microsoft.PowerShell
```

### Docsy Theme Prerequisites

* ✅ Must use Hugo Extended (already setup) 🎉
* Install PostCSS by running the following commands:

  ```sh
  npm install --save-dev autoprefixer
  npm install --save-dev postcss-cli
  npm install -D postcss
  ```
